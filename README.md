# Trabajo3


Sistema de gestión de clientes para gimnasio desarrollado en C# WinForms.

## Arquitectura
- `Trabajo3/` — Capa de presentación (WinForms)
- `Trabajo3.Data/` — Capa de acceso a datos (Repository pattern)

## Tecnologías
- C# .NET Framework
- SQL Server
- WinForms

## Configuración
1. Clona el repositorio
2. Copia `app.config.example` como `App.config` dentro de `Trabajo3/`
3. Cambia el connection string por tu servidor local
4. Compila y ejecuta.
5. Puedes agregar o crear tu DB
